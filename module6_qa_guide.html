XS + Vinchin 연동 백업 구성 절차

1
Vinchin Backup Server 설치 & 초기 설정
2
XenServer Pool 연결 — Pool Coordinator IP + 인증
3
백업 대상 VM 선택 & 그룹핑
4
백업 Job 구성 — Full+Inc 전략, SpeedKit(기본), 스케줄, Retention
5
보안 설정 + 백업 실행 + 모니터링 + 복구 테스트
전체 구성 절차를 요약합니다. Agentless로 VM에 아무것도 설치하지 않고 하이퍼바이저 API로 백업합니다.

XenServer 연동 시 주의사항

1
Pool Coordinator 연결
Vinchin은 Coordinator IP로 연결 → Coordinator 변경 시 재설정 필요
2
XenTools 필수
백업 대상 VM에 XenTools 설치 확인 (Live Snapshot 전제조건)
3
라이선스 분리
XenServer 라이선스와 Vinchin 라이선스는 별개 계약
4
DR 사이트
Vinchin 무료 DR 라이선스 활용 가능 (복구 전용)
5
백업 시간대
Pool Coordinator 변경 작업과 백업 스케줄이 겹치지 않도록 주의
5가지 주의사항을 반드시 기억하세요. 특히 Coordinator 변경과 백업 스케줄 충돌에 유의해야 합니다.
